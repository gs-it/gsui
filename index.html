이벤트 HTML 템플릿

1. 가로 940 템플릿

  • 가로폭 940px 기본 템플릿
  • #wrap 아래 이벤트 div 생성
<body>
	<div id="header"></div>
	<div id="gs-content">
		<div id="wrap">
			<div><!-- 이벤트 div 생성 -->
				이벤트
			</div>
		</div>
	</div>
	<div id="footer"></div>
</body>

2. 가로 와이드 템플릿

  • 가로 100% 이벤트 템플릿
  • #wrap 밖 #gs-content 아래 div 생성
  • .event-common-wrap을 사용 가운데 정렬(940px)
<body>
	<div id="header"></div>
	<div id="gs-content">
		<div id="wrap"><div>
		<div>
			<div class="event-common-wrap"><!-- 이벤트 컨텐츠 정렬 클래스 -->
				이벤트
			</div>
		<div>
	<div>
	<div id="footer"></div>
</body>

3. 이벤트 상단 탭

  • #wrap 아래 .wide_tabpcEvent 추가
  • .wide_tabpcEvent 하단 마진 만큼 생성한 이벤트 영역에 음수 마진 적용
  • 와이드 이벤트의 경우 생성한 이벤트 div는 #wrap 밖 #gs-content 아래 생성
<body>
	<div id="header"></div>
	<div id="gs-content">
		<div id="wrap">
			<div class="wide_tabpcEvent">
				<ul>
					<li class="first-child "><a href="#"><strong>리얼멤버십</strong></a></li>
					<li class="second-child "><a href="#"><strong>할인쿠폰</strong></a></li>
					<li><a href="#"><strong>적립금 GS&POINT</strong></a></li>
					<li><a href="#"><strong>연간할인권</strong></a></li>
					<li><a href="#"><strong>제휴신용카드 혜택</strong></a></li>
				</ul>
			<div>
			<div style="margin-top:-11px;"><!-- 탭 하단 마진 만큼 음수 마진 적용 -->
				이벤트
			</div>
		<div>
	<div>
	<div id="footer"></div>
</body>

4. 이벤트 하단 기획전 추가

  • #wrap 바로 다음 #layBody를 개발에서 추가
  • #prdlist(기획전 상품 리스트) 위치지정
  • 와이드 이벤트의 경우 생성한 이벤트 영역은 #wrap 밖 #gs-content 아래 생성
  • 점차 사용하지 않는 추세
    • <body>
	<div id="header"></div>
	<div id="gs-content">
		<div id="wrap">
			<div id="layBody"></div>
			<div><!-- 이벤트 div 생성 -->
				이벤트
			</div>
			<div id="prdlist">기획전 상품 리스트</div>
		<div>
	<div>
	<div id="footer"></div>
</body>
