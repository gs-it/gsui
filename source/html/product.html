2. 상품 모듈 - 가로폭 조정(740px)

  • 상품 모듈의 부모 클래스('goods')에 'w940'클래스를 적용한다.
  • 검색 결과등의 가로폭 740px의 경우 적용한다.

3. 상품 모듈 - 상품 이미지 상단 배지 적용

  • 상품 모듈을 기반으로 상품 이미지 하단에 해당 배지를 적용한다.
  • 상단 배지는 1개의 배지만 적용한다.

	
	Best1
	Best Deal No.1

4. 상품 모듈 - 상품 이미지 하단 배지 적용

  • 딜 상품 기본 모듈을 기반으로 상품 이미지 하단에 해당 배지를 적용한다.
  • 2개 이상의 배지를 적용할 때 'badge-pos'클래스로 태깅후 위치 지정 클래스를 조합해 적용한다.
  • 위치 지정 클래스 'bl'은 하단 좌측(bottom left) 'br'은 하단 우측('bottom right')를 의미한다.

  • 상품 가격 하단의 텍스트 기반 정보로 색상 클래스를 적용해 사용한다.
  • 색상 정보는 'red'와 'green' 두 색상을 제공한다.
