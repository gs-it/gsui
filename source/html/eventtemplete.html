가로 와이드 템플릿

  • 가로 100% 이벤트 템플릿
  • #wrap의 형제요소(#gs-content의 자식요소)에 이벤트 영역(div)를 생성해 사용한다.
  • 생성한 영역(div)에 event-common-wrap 클래스를 사용해 가운데 정렬(940px)를 적용한다.

이벤트 하단 앱다운

  • 가로 100%의 경우 #wrap의 형제요소로 #wrap 다음에 위치한다.
  • 가로 100%가 아닌 경우 가로 940이벤트와 마찬가지로 #wrap 자식요소로 생성한다.
  • CSS는 http://image.gsshop.com/mi13/css/event.css를 참조한다.
  • 앱다운 이미지 배경은 이벤트마다 변경되므로 배경은 인라인으로 적용해 사용한다.
  • 2015년 10월 현재 앱다운 영역의 높이는 418px로, 변경된다면 CSS 수정이 필요하다.
