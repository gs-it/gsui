Images Guide

이미지 기본 정책

  • 기본 포맷은 GIF, JPG 로 한다.
  • 투명 이미지일 경우에만 PNG를 사용하고 가급적 사용을 지양한다.
  • JPG는 상품 이미지 또는 인물이나 실사 이미지와 같이 색 변화가 풍부한 경우 사용한다.
  • 장식을 위한 이미지 요소들은 스프라이트 기법을 활용하여 파일을 최적화한다.
  • 반드시 이미지 최적화 툴을 사용하여 이미지 압축을 진행한다.

이미지 명명규칙

  • 이미지명은 '접두어_접속어_접속어_..'를 사용하여 이미지명으로 이미지를 유추 가능 하도록 작성한다.
  • 접두어는 이미지 접두어표의 적용순위를 기반으로 작성한다.
  • 접속어는 이미지의 상태를 표현할 수 있는 영단어를 사용한다.
  • 같은 분류의 이미지가 두 개 이상이면 파일 이름 끝에 숫자를 표기하여 구분한다
  • 임시 이미지는 접두어 'tmp_'를 사용한다. 예) tmp_
  • CSS내 이미지 경로는 상대 경로를 기본으로 한다.

더미 이미지(상품 임시 이미지)

  • 더미 이미지의 'alt' 속성 값은 공백으로 한다.
  • 더미 이미지의 'width'와 'height'속성은 사용하지 않는다.
  • 원하는 더미 이미지가 없을 경우 가까운 사이즈의 더미 이미지에 마크업 사이즈를 적용하여 사용한다.
